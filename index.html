Keyboard Accessibility Features

This website was designed to be accessible for keyboard users. Here are the main features:

  • All of the navigation links and buttons can be accessed using the Tab key. Press Tab to move forward and Shift + Tab to move backward through the page.
  • The focus indicator visually highlights links and buttons with a custom border when tabbing.
  • The “Sort List” button can be activated with the Enter or Space key when tabbing.
  • Keyboard users can press the 'S' key as a shortcut to activate the sort function.
  • All of the links on the “Things to Read” page are accessible using the Tab key.

How This Follows the Usability Principles

  • Affordance: This follows the usability principle of affordance because the "Sort List" button clearly looks and behaves like a button. The shape and label of the button tells the user what it does. The user perceives the button to sort the list of intersts. When the user clicks on it, it will sort the list. This matches the user's expectation and helps them understand how to use it by simply just looking at it.
  • Visibility: This follows the usability principle of visibility because the focus indicator clearly outlines the items when tabbing to help the user see where they are on the page.
  • Feedback: This follows the usability principle of feedback because when the "Sort List" button is clicked or activated with the 'S' key, the list immediately sorts. This change on the screen gives the user clear feedback that their action worked. They know that the list was sorted because they can see the order right away.
  • Constraints: This follows the usability principle of constraints because the website limits user interaction in a helpful way. For example, the "Sort List" button and pressing the 'S' key only works to sort the list of interests and not on any other parts of the page. This helps prevent confusion for the user.
  • Mapping: This follows the usability principle of mapping because users can navigate links and buttons using the Tab key and shortcuts. There is a clear and expected relationship between pressing Tab and moving through the items on the page. Pressing the 'S' key to activate the sort function is also a logical mapping because 'S' stands for sort, so it makes sense and is easy for the user to remember what the key does.
  • Consistency: This follows the usability principle of consistency because all of the links and buttons behave the same way on all of the pages. This makes the website easier to use and understand.
