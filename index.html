Keyboard Accessibility Features

  • All of the navigation links and buttons can be accessed using the Tab key.
  • The focus indicator visually highlights links and buttons with a custom border when tabbing.
  • The “Sort List” button can be activated with the Enter or Space key when tabbing.
  • Keyboard users can press the 'S' key as a shortcut to activate the sort function.
  • All of the links on the “Things to Read” page are accessible using the Tab key.
